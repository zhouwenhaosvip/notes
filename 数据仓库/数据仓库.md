## 一.数据库、数据仓库、数据空间

| 阶段         | 面向对象 | 解决问题               | 技术代表                           |
| ------------ | -------- | ---------------------- | ---------------------------------- |
| **数据库**   | 业务运行 | 记录业务数据（OLTP）   | MySQL / Oracle                     |
| **数据仓库** | 分析决策 | 分析海量数据（OLAP）   | Hive / Spark / ClickHouse          |
| **数据空间** | 跨组织   | 数据共享、治理、资产化 | Data Space / DataMesh / DataFabric |
